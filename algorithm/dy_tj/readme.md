### 双塔模型广撒网
千人千面
千万级视频 快速筛选 
- 第一步 召回：双塔模型
向量的点积运算，cosin夹角，把相关的1000条视频筛选出来
- 第二步 初排（粗）排
用相对轻量的机器学习模型，算一些粗的内部特征，从上面的1000条，筛选为300条 
- 第三步 精排
预测点不点击，完播的概率 点赞，评论，分享 score=w1.点击*  w2.完播*  w3.点赞*  w4.评论*  w5.分享
- 重排
如果前十名都是你爱看的视频，重排机制会强制打散。每两个硬核内容，强制加入一个其他视频，防止产生疲劳
