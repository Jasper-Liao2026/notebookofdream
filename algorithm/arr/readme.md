## 数组的创建

- ADT的认识
    连续的存储空间+特定的操作
- push,pop,shift,unshift
    - 修改原数组 破坏了原来数组，这些方法都不是纯函数
    - push 返回结果是新长度
    - pop 移除的最后一个元素的值
    - unshift 在对头插入元素，
- 纯函数
- new Array(7)
指定了长度的空数组[empty*7]  意思是数组的这个位置还没有被占据，不属于任何一个类型
此时arr[0] 未设置则为undifined
- new Array(7).fill(1)
创建一个长度确定，同时每一个元素也确定的数组

## 遍历的方法
- for 技术
    机器化，命令式 缺点：可读性不好  优点：性能最好
- for of 
    语义很好
- forEach
    入调用栈 执行上下文....
    优点是功能强大
    缺点是不能中途break;
- map、filter、every、some
    基于forEach
    返回一个全新数组 纯函数
    filter 筛选 函数结果为true留下
    every 每一项满足结果，true
    some 有一项返回 ，true
