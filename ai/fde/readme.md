# FDE Forward Deloyed Engineer

最近AI圈有个岗位很火，不是算法工程师，也不是agent工程师，而是fde（前沿部署工程师）

前沿部署工程师
Soft Engineer(古法软件工程师 copilot代码建议 人为主力 AI辅助)  -> 
prompt engineer(提示词工程师 AI为主) ->
agent engineer(智能体工程师 AI编程能力强 可靠地执行) ->
Forward Deployed Engineer(业务结合)

## FDE 公司
Open AI成立了一个FDE公司，tomoro
Anthropic 和 高盛 黑石 成立了ai服务公司,把 Claude 带进企业和核心业务流程，数据化 数据智能

AI 公司已经进入一个新的阶段 ，过去大家拼的是模型能力，谁参数更强，谁上下文更长(一次可以给llm的大小)，谁跑分更高(计算，推理，编程)，谁写代码更厉害  ————————    现在是谁真正把ai落地在企业业务里更强

企业买ai 不是买一个api 也不是买token，企业要降本、提效、增长、风控、一个能真正嵌入业务流程、每天被员工使用，最后能产生业务结果的系统。

FDE更像是站在客户现场的ai工程负责人

工作内容：
- 理解客户业务流程，越垂直越好
- 哪些环节适合ai来改造
- 把企业内部数据、系统、工具链接起来
- 搭建Agent、工作流、知识库和评测体系
- 确保这些东西在生产环境跑起来

从以前的卖模型到卖结果
