# 树与二叉树

数据结构的树，是对现实世界中树的一层简化
- 树根 根节点，
- 树枝是边
- 树枝的两端为节点
- 树叶抽象为叶子节点

## 定义
- 他可以没有根节点，作为一棵空树存在
- 如果不是空树，那么必须由根节点，左子树和右子树组成，且左右子树都是二叉树

## 树的关键概念
- 树的层次计算规则
    根节点所在的那一层为第一层，其子节点在第二层，类推之
- 树的高度
    叶子节点高度计为1，每向上一层高度加一
- 度的概念
    一个节点开叉出去多少给子树，节点的度
- 叶子节点
    度为0的节点，最后一层都是叶子节点

二叉树不可以被简单定义为每个节点的度为2的树
左右子树的位置是严格约定，不能交换的
