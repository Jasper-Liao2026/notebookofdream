# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目概述

这是 Claude Code 和 Web 前端开发的学习工作区，用户在此学习使用 Claude Code 进行 AI 辅助编程。所有项目均为纯静态前端页面，无需构建工具或框架。

## 环境

- npm 镜像源已配置为淘宝源：`registry https://registry.npmmirror.com`
- Claude Code 已全局安装（`@anthropic-ai/claude-code`）
- 无 git 仓库、无构建系统、无包管理器配置文件

## 项目结构

- `jiama/` — Web 前端项目目录
  - `index.html` + `css/style.css` + `js/main.js` — 吉马程序员品牌宣传单页，纯 HTML/CSS/JS
  - `2026_05_21_style/` — JS Drum Kit 练习项目（JavaScript30 第一课），通过键盘按键触发架子鼓音效
- `readme.md` — Claude Code 与 Vibe Coding 学习笔记

## 开发方式

- 所有页面为纯静态文件，直接在浏览器打开 HTML 文件即可预览
- 无需 `npm install`、`npm run build` 或 `npm test`
- 用户偏好现代简约设计风格（modern minimalist）
